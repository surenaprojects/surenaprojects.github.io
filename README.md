## 📖 About

This repository hosts the source code for my personal portfolio, built and deployed with **GitHub Pages**. It's a single-page site presenting who I am, what I do, and the projects I've built — with a clean, developer-themed (terminal-inspired) design.

I'm passionate about coding, reading, and bodybuilding, currently deep into the **Linux** ecosystem, and always sharpening my craft in **Python**, **Dart**, and **JavaScript**.

## ✨ Features

- 🖥️ **Terminal-inspired UI** — `whoami`-style section headers for a developer feel
- 🌍 **Multi-language support** — English, 中文 (Chinese), Русский (Russian), Español (Spanish)
- 📱 **Fully responsive** — built with Tailwind CSS for a seamless experience across devices
- 🧭 **Single-page navigation** — Home, About, Skills, Projects, Contact

## 🚀 Getting Started

Clone the repository and open it locally:

```bash
git clone https://github.com/surenaprojects/surenaprojects.github.io.git
cd surenaprojects.github.io
```

Then simply open `index.html` in your browser, or serve it locally:

```bash
# Using Python
python -m http.server 8000

# Then visit
http://localhost:8000
```
